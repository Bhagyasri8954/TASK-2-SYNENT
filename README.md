# Number guessing game
A simple and interactive command-line number guessing game built using Python.
# Description
The program generates a random number, and the user has to guess it within a limited number of attempts.
After each guess, the program provides hints to help the user.
# Features
* Random number generator
* Multiple attempts
* Hint system(Too low/Too high)
* Attempt counter
* Game ends when correct guess or attempts finish
* Handles invalid input gracefully
# Technologies used 
Python3
# How to run 
python numberguess.py
